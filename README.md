# Spark ETL Pipeline Project

## Description

Projet réalisé dans le cadre du Jour 4 de la formation Apache Spark.

L'objectif est de concevoir un pipeline ETL complet avec PySpark :

- Ingestion des données (Bronze)
- Nettoyage et transformation (Silver)
- Analyses métier (Gold)
- Optimisation des performances
- Exploration avancée
- Rapport technique

## Technologies

- Python 3.12
- Apache Spark 4.0.1
- PySpark
- Git

## Architecture

```
data/
    raw/
    silver/
    gold/

src/
    ingestion.py
    transform.py
    analysis.py
    optimization.py
    exploration.py
    utils.py
```
